Allow WordPress to send email via SMTP.

== Description ==

A no thrills approach to allowing WordPress to send email via SMTP. Once enable and configured you can start sending email via Gmail.

== Settings ==

Once activated you can enable WP MAIL SMTP ZM from the WordPress settings menu. It is accessible via the sub-menu WP MAIL SMTP ZM. Enable it. Add your settings. Done.

Once enabled mail will be sent using your SMTP settings.

== Standard Features ==

* Custom from email
* Custom from name
* Choose mailer
* Set the return path
* Send test email
* Gmail pre-sets

=== SMTP Settings ===

The SMTP settings allow you to set the custom options for your host.

* Port
* Encryption Type
* Authentication
* Username
* Password

**If additional settings are needed for your specific email provide. Please ask and they will be added**

== Installation ==

= Automatic =

1. Go to Plugins > Add New.
1. Type in the name of the WordPress Plugin "WP MAIL SMTP ZM"
1. From the search results find "WP MAIL SMTP ZM"
1. Click Install Now to install the WordPress Plugin

= Manual =

1. Upload the `quick-gallery` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
